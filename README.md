# xcheck
To run the app in the development mode:
### npm run dev
or
### yarn dev
To view applicaton in the browser open http://localhost:3000

### Technologies in the project :
- React
- Typescript
- Next.js
- JSON-server
- Ant Design
- Github OAuth
- Firebase
- Formik

### Roles:
